We're excited to share the latest features that have been rolled out in January and February! This update focuses on improving field worker experience with better navigation capabilities, enhanced issue reporting, and significant performance improvements for data uploads.

Let's explore what's new in this release.

### üìç Location Navigation

Field workers can now easily save and navigate to beneficiary locations. When at a beneficiary's residence, field workers can click the add location icon to save the beneficiary's precise location. Once saved, they can tap the location to navigate back to it anytime.

With offline maps downloaded, field workers can navigate to saved locations even without an internet connection‚Äîensuring uninterrupted fieldwork in remote areas.
